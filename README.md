# Pinboard

# Background
Pinboard is a Pinterest clone built on react and rails.

# Features
In Pinboard, users are able to:
- create, edit, and delete Pins
- create, edit, and delete Boards that contain Pins

# Technologies, Libraries, and APIs
- React, rails, AWS S3
